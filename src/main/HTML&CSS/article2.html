Madspildets Pris



Sidst opdateret d. 05/11/2003
Skrevet af: Joakim Brodén

Vi sløser væk, vi bruger op, vi smider ud, det er en frygtelig krig,
Madspildets præmie er høj, og vi kan ikke fortsætte sådan her.
Der er intet respekt for de ressourcer, som gik til at producere maden,
Det er på tide, vi står sammen og ændrer vores vaner i stedet for at skade planeten.

Spildt Mad, Spildte Ressourcer

For hvert stykke mad, vi kaster væk, går det tabt,
Tabt vand, jord og energi, som vi aldrig får tilbage igen,
Det er en tragisk kamp, vi kæmper, hvor taberne er mange,
Men hvis vi ændrer vores vaner, kan vi skabe en sejr, der varer længe.

En Skammelig Kamp

Vi har en hær af producenter, en hær af distributører,
Men vi har også en hær af forbrugere, der ikke tager ansvar,
Vi kan ikke længere se på, mens denne krig raser videre,
Vi må stå op, tage kontrol og ændre denne skammelige kamp for evigt.

Den Nye Kamp

Vi kan begynde med små skridt og langsomt ændre vores måde at handle på,
Spar på maden, køb kun, hvad vi har brug for, og planlægning er vejen frem,
Lad os opfordre vores lokale producenter til at reducere deres spild,
Vi kan tage ansvar og vinde denne nye kamp, så fremtiden bliver lys og mild.

Så lad os stå sammen, og lad os kæmpe for en bedre fremtid,
Lad os reducere madspild og tage ansvar for vores handlinger,
Lad os ændre vores vaner og vinde denne kamp, der raser,
For sejren vil være sød, når vi kan sige, at madspildets pris blev betalt.
